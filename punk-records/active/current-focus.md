# Current Focus

Building Punk Records — the living brain MCP server that Claude Code can tap into and contribute to. Inspired by Vegapunk's Punk Records from One Piece.

## What's Happening Now
- MCP server built with 18 tools, 5 prompts, 6 resources
- Hybrid search (semantic + keyword) via Transformers.js + Orama
- Knowledge graph with JSONL-backed entity/relation store
- Query logging and analytics for proactive suggestions
- Knowledge base being seeded with initial data from architecture document

## Next Steps
- Real-world testing with Claude Code across actual projects
- Populate knowledge base with more project-specific content as work happens
- Refine search quality based on actual usage patterns
- Build out the knowledge graph with more relationships as patterns emerge
