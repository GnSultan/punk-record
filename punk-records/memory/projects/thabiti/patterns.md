# Thabiti — Reusable Patterns

## B2B Brand Engagement Pattern
1. Start with why workshop (client often skeptical — prove value)
2. Map customer's actual fears, not just needs
3. Position brand as guide using StoryBrand
4. Visual identity reflects local context
5. Messaging framework before any design work

## Key Reusable Insight
- Tanzanian B2B clients need to see the "why" of branding before buying in
- Start with a concrete problem they recognize, not abstract brand theory
- Show how competitors' lack of positioning costs them customers
