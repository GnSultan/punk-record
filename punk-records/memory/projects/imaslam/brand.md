# imaslam — Personal Brand Identity

## What This Is

The personal brand and portfolio at imaslam.com. This is how Aslam presents his freelance craft to the world — brand strategy, design, and development as one connected thread.

This brand evolved from Pulse Brand Studio. Pulse was an agency name, but it was only one person with no companion. As the work deepened and authenticity became non-negotiable, the agency model felt misleading. Quit it and became a simple freelancer under his own name. The methodology carried over. The honesty is what changed.

---

## Positioning

**What he actually does:** Takes a client from "I need a brand" through strategy, visual identity, and into a working product. The full thread — why it should exist, how it should feel, how it gets built. Not a designer who codes or a developer who designs. Someone who does the whole thing because the parts shouldn't be disconnected.

**Tagline:** Start with why, design with empathy, build with precision.

This isn't marketing copy. It's literally the order of operations. Every project starts with why (discovery meeting, deep questions about purpose). Then empathy (understanding the real audience, their context, their constraints). Then precision (strategy → visuals → code, nothing wasted).

## What the Brand Stands For

These aren't values invented for a brand book. They're how the work actually happens:

1. **Start with why** — No project begins without understanding purpose. The discovery meeting comes before any design or technical discussion.
2. **Strategy before visuals** — The visual identity is derived from strategy, never arbitrary. Nine phases, strategy locked before any pixel is placed.
3. **Authenticity** — Real identities, not staged ones. If a client can't participate in building their own brand, the project gets terminated. Makando Travel proved this principle.
4. **Quality through work, not words** — The portfolio itself is the proof. Monochromatic palette demonstrates design skill. Smooth animations demonstrate technical skill. Strategic copy demonstrates thinking. Everything is evidence.
5. **Depth over decoration** — Branding is not a logo and color palette. It's a system of meaning. Clients who expect a takeout order don't last.

## Visual Identity

### Color Palette — Monochromatic

No accent colors. The palette is 4 values — everything else is typography and motion. This is a deliberate choice: prove the design skill without relying on color. If the layout, typography, and motion hold without color, the design is real.

**Light Mode:**
| Role | Hex | Description |
|------|-----|-------------|
| Background | #FAFAFA | Off-white |
| Text / Primary | #0F0F0F | Near-black |
| Text Secondary | #2A2A2A | Dark gray |
| Secondary (UI) | #E5E5E5 | Light gray |
| Footer BG | #000000 | Pure black |
| Footer Text | #FFFFFF | Pure white |

**Dark Mode:**
| Role | Hex | Description |
|------|-----|-------------|
| Background | #0F0F0F | Near-black |
| Text / Primary | #FAFAFA | Off-white |
| Text Secondary | #B0B0B0 | Medium gray |
| Secondary (UI) | #2A2A2A | Dark gray |
| Footer BG | #FFFFFF | Pure white |
| Footer Text | #0F0F0F | Near-black |

Footer fully inverts for dramatic contrast.

### Typography — Clash Grotesk

Single typeface, weight variation creates all hierarchy:
- **200-300**: Accents, lighter hierarchy
- **400**: Body text (1.5rem, line-height 1.7)
- **500**: Secondary emphasis, subheadings
- **600-700**: Headlines (up to 7rem, line-height 0.9, letter-spacing -0.03em)

All sizing uses `clamp()` for continuous fluid scaling — no breakpoint jumps.

### Visual Motifs

**The Asterisk** — A geometric 8-pointed star constructed from intersecting lines. Rotates on scroll. Carries across projects (also used in Thabiti). Becoming a signature element — abstract, adds character without being literal.

**Grid Overlay** — 1px vertical lines on the background (80px mobile, 200px desktop, 4-8% opacity). Creates subtle visual structure.

**Full-width 1px Dividers** — Animated with scaleX from 0 to 1. Clean section separation.

### Layout Philosophy
- 70/30 content splits for visual hierarchy
- Generous whitespace — `clamp(4rem, 8vw, 8rem)` between sections
- Content constrained to 65 characters for readability
- Massive headlines dominating viewport, small supporting text below
- Empty space is intentional — it *is* the design

### Motion as Architecture
- Lenis smooth scrolling (1.5s, cubic easing)
- Framer Motion scroll-triggered reveals (0.6s, stagger)
- Parallax depth on images (speed -0.5 to -0.8)
- Custom cursor with spring physics, magnetic attraction, context labels
- WebGL fluid background with Perlin noise
- Grayscale images → full color on hover
