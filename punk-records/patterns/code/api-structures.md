# AI-Powered Lesson Generation

**Description:** Using LLMs to generate educational content that adapts to curriculum and student level.

## Implementation Approach
- Structured prompts with curriculum context
- Generation in Swahili-native, not translated
- Auto-marking with detailed feedback
- Difficulty adjustment based on performance
- Real-time teaching whiteboard integration

## Key Challenges
- Ensuring curriculum alignment
- Quality control at scale
- Offline caching of generated content
- Cost management for API calls

## When to Use
- Educational platforms needing content scale
- Personalized learning paths
- Practice problem generation

## When to Avoid
- When expert human review isn't possible
- Sensitive topics requiring human judgment
- When curriculum is highly standardized with existing content

## Open Questions
- How to validate educational accuracy at scale?
- What's the right balance of AI-generated vs. human-created content?

## Applied In
- ElimuAfrica (lesson and assessment generation)

**Success Rate:** 75% — promising but still evolving. Confidence: 70%.
